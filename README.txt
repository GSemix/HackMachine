Структура logs.txt:

    Start in <time>

	<...>

	Что случилось:
	{ [+] Успешно выполнена команда        }
	{ [-] Ошибка при выполнении!           }
	{ <...> Информация о состоянии системы }

	<...>

    Stoped in <time>
